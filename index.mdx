    <h2 className="text-3xl font-bold dark:text-white text-gray-900 mb-4">Get started in minutes</h2>
    <p className="text-lg dark:text-gray-400 text-gray-600">Follow these steps to install and configure IronClaw</p>
  </div>

  <Steps>
    <Step title="Install IronClaw">
      Install IronClaw using your preferred package manager or download the binary directly:

      <CodeGroup>
      ```bash Shell script (macOS/Linux)
      curl --proto '=https' --tlsv1.2 -LsSf https://github.com/nearai/ironclaw/releases/latest/download/ironclaw-installer.sh | sh
      ```

      ```powershell PowerShell (Windows)
      irm https://github.com/nearai/ironclaw/releases/latest/download/ironclaw-installer.ps1 | iex
      ```

      ```bash Homebrew (macOS/Linux)
      brew install ironclaw
      ```

      ```bash Cargo
      cargo install ironclaw
      ```
      </CodeGroup>
    </Step>

    <Step title="Set up your database">
      IronClaw requires PostgreSQL with the pgvector extension for persistent storage:

      ```bash
      # Create database
      createdb ironclaw

      # Enable pgvector extension
      psql ironclaw -c "CREATE EXTENSION IF NOT EXISTS vector;"
      ```

      <Note>
        You can also use libSQL/Turso as an embedded database alternative. See the [database configuration guide](/configuration/database) for details.
      </Note>
    </Step>

    <Step title="Run the onboarding wizard">
      Configure IronClaw with the interactive setup wizard:

      ```bash
      ironclaw onboard
      ```

      The wizard will guide you through:
      - Database connection setup
      - LLM provider authentication (NEAR AI by default)
      - Secrets encryption configuration
      - Optional channel setup (Telegram, HTTP, etc.)
    </Step>

    <Step title="Start using IronClaw">
      Launch the interactive REPL to start chatting with your assistant:

      ```bash
      ironclaw run
      ```

      You can also use single-message mode for quick queries:

      ```bash
      ironclaw -m "What's the weather like today?"
      ```
    </Step>
  </Steps>
</div>
